Enterprise Infrastructure Deployment

Structured Cabling & Hierarchical Network

Engineered a scalable hierarchical star backbone routing Category 6 and OM4 Fiber Optics from a central Equipment Room (ER) to multiple localized Telecommunications Rooms (TRs) and Enclosures (TEs).

Operational Objectives
  • Deliver standards-compliant, future-proof cabling infrastructure.
  • Support high-speed data, voice, and video applications.
  • Create a scalable, easily maintainable network backbone with excellent fault isolation.
Technical Execution
Hierarchical Topology

Interconnected individual star topologies. Established backbone runs consisting of 6 Cat 6 cables and 2 fiber optic cables from ER (North) to TR East, TR West, and TE South through a central conduit cross box.

Copper & Fiber Implementation

Installed and terminated Cat 6 UTP (T568A/B configs) and coaxial (F-connector). Replaced legacy OS1 with high-bandwidth OM4 Multimode Fiber. Terminated all horizontal and backbone cabling at patch panels for organized cross-connects.

Compliance & Standards

Performed all work in strict accordance with ANSI/TIA-568 guidelines and BICSI testing standards to ensure top-tier performance, safety, and code compliance.

Systems Engineering & Hardware Repair

Dell Server Hardware & OS Restoration

Comprehensive root-cause analysis and remediation of a severe boot loop failure, encompassing hardware isolation and operating system integrity repair.

Issue: Server stuck in boot loop/hang state with No POST and No VGA Output.
Root Cause: Defective RAM module causing POST failure, leading to Windows Server OS file corruption.
1. Initial Identification

Remote checks via IPMI confirmed the system was receiving power but failing to initialize video output. Proceeded to physical hardware diagnostics.

2. Hardware Isolation

Tested CMOS battery (nominal voltage). Proceeded to reseat memory modules. Intermittent video return indicated a hardware fault, not a connection issue.

3. Fault Isolation & POST Success

Systematically removed RAM sticks to isolate the defective component. Successfully identified one faulty module. Server POSTed with stable video output (operating at 2GB for lab constraints).

Operating System Restoration

Challenge: Repeated failed boot attempts corrupted Windows Server OS files, halting boot sequence even after hardware repair.

  • Accessed the Windows Recovery Environment (WinRE).
  • Executed built-in Startup Repair to rebuild Boot Configuration Data (BCD).
  • Executed a Safe Mode (Network Enabled) boot to verify OS integrity before a standard restart.
Final Verification
  • Hardware: Passed POST, stable output.
  • Software: Windows Server boot sequence successful.
  • Network: Fully operational; login halted intentionally due to isolated Active Directory (AD) lab configuration unreachable from current domain.
